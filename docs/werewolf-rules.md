## 1. 概述

狼人杀是一款社交推理类桌游。玩家被秘密分配角色，分属好人阵营或狼人阵营。
游戏在夜晚（狼人秘密行动）和白天（公开讨论与投票）之间交替，直到某方获胜。

- 标准竞技格式：12 名玩家 + 1 名法官（主持人，不参与胜负）。
- 法官负责：分发身份牌、主持夜晚唤醒、公布死讯、管理发言与投票、判定胜负。

---

## 2. 阵营与胜利条件

### 2.1 好人阵营

包含：村民（普通平民）+ 神职（预言家、女巫、猎人、白痴/守卫/骑士）

胜利条件：所有狼人出局。

### 2.2 狼人阵营

包含：狼人牌（人数由所选板子决定）。

胜利条件（屠边模式，主流竞技）：

- 所有神职出局（屠神），或
- 所有村民出局（屠民）

胜利条件（屠城模式，经典休闲）：

- 所有好人阵营成员出局（神职 + 村民全部死亡）

### 2.3 第三阵营：情侣（可选，需丘比特角色）

丘比特首夜连接一对情侣，一方死亡则另一方殉情。
情侣胜利条件：场上仅剩情侣二人存活。标准竞技中较少使用。
说明：本规则正文未展开丘比特的完整行动流程；若启用情侣请在开局配置中补充丘比特规则。

### 3.7 板子平衡性说明

- 预女猎白：基础板，好人阵营略占优势，适合新手。
- 预女猎守：守卫能力强，需搭配功能狼（白狼王/狼美人）平衡。
- 预女猎骑：骑士决斗能力强大，同样需搭配功能狼平衡。
- 含功能狼的板子节奏更快，策略更丰富，为当前竞技主流。

通用比例：约每 3 名玩家配 1 名狼人。

### 3.8 开局前配置锁定（引擎必需）

- 必须在开局前明确并锁定本局配置：几人局、具体板子、胜利模式、女巫自救规则、是否启用情侣。
- 本文档中的 9 人局、12 人局各板子为互斥配置；单局只能选择其一，不允许中途切换。
- 若启用情侣，必须同时启用丘比特并提供对应角色配置；本节 3.1-3.6 未包含丘比特扩展板。
- AI 引擎应在 `game_config` 中显式记录以上参数，并在首夜前向所有玩家公告可公开项。
- 若未锁定配置，法官不得开始发牌与夜间流程。

## 4. 角色技能详解

### 4.1 村民

- 没有任何特殊技能。
- 白天可以发言和投票。
- 屠边模式下村民战略价值高：可伪装神职来吸收狼刀（挡刀策略）。

### 4.2 狼人

- 夜晚行动：
  - 所有狼人一起睁眼，互相确认身份。
  - 统一选择一名玩家击杀。
  - 允许空刀（不杀人）。
  - 允许自刀（杀自己队友或自己）。
  - 若无法统一意见，视为空刀。
- 白天行动：
  - 正常发言和投票，伪装好人身份。
  - 可以在投票前选择自爆（详见 12.5 自爆规则）。
- 信息优势：狼人互相知道彼此身份，好人不知道。

### 4.3 预言家

- 夜晚行动：选择一名存活玩家进行查验。
- 查验结果：法官指示该玩家是"好人"还是"狼人"。
  - 只能判断阵营，无法得知具体角色。
  - 拇指向上 = 好人，狼爪手势 = 狼人。
- 每晚查验一人，不能查验已死亡的玩家，不能查验自己。

### 4.4 女巫

拥有两瓶一次性药水（整局各一瓶）：

**解药（救人）：**

- 救活当晚被狼人刀杀的玩家。
- 使用时机：法官展示被刀者后，女巫决定是否使用。
- 解药用完后，法官不再向女巫展示被刀者信息。
- 解药用尽后，法官仍需以固定时长执行女巫阶段（仅不给信息），防止时长差异泄露信息。

**毒药（杀人）：**

- 毒死任意一名存活玩家（不需要是被刀者）。
- 使用时机：与解药同一行动阶段，但不能同夜使用两瓶药。

**使用限制：**

- 同一晚不能同时使用解药和毒药。
- 自救规则（三种常见版本，开局前需确认）：
  - 全场不可自救（主流竞技规则）
  - 仅首夜可自救
  - 全场可自救
- 解药用完后，女巫不再获知每晚被刀者身份。
- 同守同救规则：若守卫和女巫同时保护被刀者，保护效果抵消，被刀者死亡。

### 4.5 猎人

**被动技能 - 开枪：**

- 猎人死亡时，可以选择开枪带走场上任意一名存活玩家。
- 也可以选择不开枪。
- 被猎人带走的玩家立即出局。

**开枪条件：**


| 死亡原因    | 能否开枪 |
| ------- | ---- |
| 被狼人刀杀   | 能    |
| 被白天投票放逐 | 能    |
| 被女巫毒死   | 不能   |
| 情侣殉情    | 不能   |


- 法官每晚会通知猎人当前的开枪状态（能/不能）。
- 被毒死时，法官会在夜间提示猎人技能无法发动。

### 4.6 守卫

- 夜晚行动：选择一名存活玩家进行守护。
- 被守护者当晚不会被狼人刀杀。
- 可以守护自己。
- 可以空守（不守护任何人）。
- 不能连续两晚守护同一名玩家。
- 守护只能抵挡狼刀，不能抵挡女巫毒药。
- 同守同救规则：守卫守护 + 女巫解药作用于同一被刀者 = 被刀者死亡。

### 4.7 骑士

**主动技能 - 决斗：**

- 骑士可在白天发言阶段（非警长竞选时）主动翻牌，选择一名玩家发起决斗。
- 若决斗目标是狼人 → 该狼人立即出局，白天结束，直接进入夜晚。
- 若决斗目标是好人 → 骑士自己立即出局，白天继续正常流程。
- 技能仅能使用一次（无论成功与否）。

**决斗时机限制：**


| 阶段         | 能否决斗 |
| ---------- | ---- |
| 白天轮次发言阶段   | 能    |
| 警长竞选发言阶段   | 不能   |
| 平票 PK 发言阶段 | 不能   |
| 遗言阶段       | 不能   |


**特殊情况：**

- 决斗功能狼（如白狼王、狼美人）成功时，被决斗的功能狼无法发动技能。
例：狼美人被骑士决斗出局，魅惑效果不触发。
- 若骑士是场上最后一个神职（屠边模式），决斗好人失败出局后，可能立即触发
狼人胜利条件（所有神职出局），此时骑士无遗言、无法移交警徽。

### 4.8 白狼王（功能狼）

属于狼人阵营。夜间与其他狼人一起行动（睁眼、指刀）。

**主动技能 - 自爆带人：**

- 白狼王在白天自爆时，可以同时带走一名玩家。
- 仅在自爆时才能发动技能。被投票放逐、被狼刀等其他方式出局时不能带人。
- 自爆后翻牌公开白狼王身份。

**自爆时机限制：**


| 阶段       | 能否自爆 |
| -------- | ---- |
| 警长竞选发言阶段 | 能    |
| 警长竞选退水阶段 | 能    |
| 白天轮次发言阶段 | 能    |
| 警长投票阶段   | 不能   |
| 放逐投票阶段   | 不能   |
| 遗言阶段     | 不能   |


**特殊情况：**

- 白狼王自爆带人后，双方均无遗言；完成胜利检查与必要的警徽处理后进入夜晚。
- 若白狼王是场上最后一只狼，自爆时不能发动带人技能。
- 若白狼王不自爆、以其他方式出局，在其他玩家视角中视为普通狼人，有遗言。
- 白狼王若在夜间已被女巫下毒（次日进入死亡名单），则次日白天不能发动自爆带人。

### 4.9 狼美人（功能狼）

属于狼人阵营。夜间与其他狼人一起行动后，单独睁眼执行魅惑。

**被动技能 - 魅惑：**

- 每晚狼人行动结束后，狼美人单独睁眼，选择一名玩家进行魅惑。
- 魅惑效果：当狼美人在当晚或次日白天死亡时，被魅惑的玩家也随之殉情出局。
- 每晚必须选择一名魅惑目标（不能空过）。
- 可以魅惑狼队队友。

**技能限制：**

- 狼美人不能自刀（狼人指刀时不能选狼美人自己）。
- 狼美人不能自爆。
- 殉情的猎人不能开枪。
- 殉情的白痴不能翻牌。
- 若狼美人被骑士决斗出局，魅惑效果不触发（被决斗的功能狼技能失效）。

**夜间唤醒顺序（含狼美人）：**
守卫 → 狼人（含狼美人）→ 狼美人单独睁眼魅惑 → 女巫 → 预言家

### 4.10 白痴

**被动技能 - 免死翻牌：**

- 若在白天投票环节被放逐，白痴翻牌亮出身份牌，免除放逐，继续存活。
- 翻牌后永久失去投票权，但保留发言权。
- 翻牌后仍可以竞选/持有警长（但无投票权）。

**技能限制：**


| 死亡原因    | 能否翻牌     |
| ------- | -------- |
| 白天投票放逐  | 能（免死）    |
| 被狼人刀杀   | 不能（正常死亡） |
| 被女巫毒死   | 不能（正常死亡） |
| 被猎人开枪带走 | 不能（正常死亡） |
| 情侣殉情    | 不能（正常死亡） |

## 5. 游戏流程总览

```
游戏开始 → 法官发牌，玩家确认身份
  │
  ▼
[第一夜] 所有角色按顺序行动
  │
  ▼
[第一天] 警长竞选 → 公布死讯(有遗言) → 发言 → 投票
  │
  ▼
[第二夜] 存活角色按顺序行动
  │
  ▼
[第二天] 公布死讯(无遗言) → 发言 → 投票
  │
  ▼
  ... （循环，直到胜利条件达成）
```

## 6. 第一夜流程

法官宣布"天黑请闭眼"，所有玩家闭眼并制造节奏噪音（敲桌子）。

### 6.1 有守卫的配置（预女猎守）


| 步骤  | 法官指令     | 行动              |
| --- | -------- | --------------- |
| 1   | "守卫请睁眼"  | 守卫选择守护目标（或空守）   |
| 2   | "守卫请闭眼"  | -               |
| 3   | "狼人请睁眼"  | 狼人睁眼互相确认        |
| 4   | "狼人请杀人"  | 狼人统一指定击杀目标      |
| 5   | "狼人请闭眼"  | -               |
| 6   | "女巫请睁眼"  | 法官展示被刀者（手势比号码）  |
| 7   | -        | 女巫决定用解药/毒药/不用药  |
| 8   | "女巫请闭眼"  | -               |
| 9   | "预言家请睁眼" | 预言家指定查验目标       |
| 10  | -        | 法官展示查验结果（好人/狼人） |
| 11  | "预言家请闭眼" | -               |
| 12  | "天亮请睁眼"  | 所有玩家睁眼          |


### 6.2 无守卫的配置（预女猎白）

跳过步骤 1-2，其余相同。即：狼人 → 女巫 → 预言家。

若使用自定义板子并启用"无守卫 + 狼美人"组合，第一夜在"狼人闭眼"后插入"狼美人单独睁眼魅惑"阶段，顺序为：狼人 → 狼美人 → 女巫 → 预言家。

### 6.3 首夜特殊说明

- 守卫在狼人之前行动，防止守卫根据狼人行为获取信息。
- 已死亡角色的行动阶段，法官仍需"假唤醒"（喊口令但无人行动），防止信息泄露。
- 法官在每个夜间阶段应使用固定口令与固定时长（建议 5 秒）；无可行动角色时也需执行等时假流程。
- 守卫首夜通常选择空守：因为同守同救规则，如果守卫首夜守了被刀者，
而女巫又用了解药，两者抵消反而害死被刀者。

## 7. 第一天流程（含警长竞选）

第一天是唯一有警长竞选的白天。流程顺序：

### 7.1 警长竞选

#### 7.1.1 上警（报名）

1. 法官宣布"所有玩家闭眼，想要竞选警长的玩家请举手"。
2. 玩家闭眼举手（防止互相影响）。
3. 法官记录举手者，宣布"天亮请睁眼"。
4. 法官公布竞选者名单。

特殊情况：

- 无人上警 → 本局无警长。
- 仅一人上警 → 自动当选。

#### 7.1.2 竞选发言

- 竞选者（警上玩家）按顺序发言。
- 发言顺序规则：根据当前时间分钟数的尾数，单数顺时针，双数逆时针。
例：15:25 → 尾数 5（单数）→ 按号码从小到大发言；15:26 → 尾数 6（双数）→ 从大到小。
- 非竞选者（警下玩家）在此阶段不发言。

#### 7.1.3 退水

- 竞选发言结束后，法官宣布"退水时间"。
- 竞选者可以选择退出竞选（退水），放弃成为警长。
- 退水后不能参与警长投票（既不能被投也不能投人）。

#### 7.1.4 警长投票

1. 仍在竞选的玩家亮出号码牌。
2. 法官宣布"警下玩家闭眼投票"。
3. 非竞选且未退水的玩家闭眼举手投票。
4. 每人只能投一票，可以弃票。
5. 得票最高者当选警长。

#### 7.1.5 警长竞选平票处理

- 两人或多人得票相同 → 进入 PK 发言。
- PK 玩家各发言一次，然后警下且未退水的玩家（不含 PK 玩家）再次投票。
- 警长竞选 PK 复投按每名合格投票者 1 票计，不存在 1.5 票或归票加票。
- 若再次平票 → 警徽流失，本局无警长。

### 7.2 警长权力


| 权力      | 说明                                   |
| ------- | ------------------------------------ |
| 1.5 票   | 警长在白天投票时票数算 1.5 票                    |
| 归票权     | 所有玩家发言后，警长可进行归票总结，警长归票某号 = 该号票数 +0.5 |
| 决定发言顺序  | 警长选择从自己左手边（警左）或右手边（警右）开始发言           |
| 警徽移交/撕毁 | 警长死亡时，可将警徽交给任意存活玩家，或撕毁警徽             |


### 7.3 公布死讯

- 法官宣布第一夜的死亡情况："昨夜 X 号玩家死亡"。
- 多人死亡时，按号码从小到大公布。
- 第一夜死亡的玩家有遗言机会。
- 若无人死亡："昨晚是平安夜"。

### 7.4 发言阶段

- 发言顺序：
  - 有警长时：一律从警长左侧或右侧开始，由警长决定方向。
  - 无警长且有夜间单死时：从死者左侧或右侧开始，由法官按开局约定方式决定。
  - 无警长且双死/平安夜时：根据当前时间分钟各位数字相加确定起始号码。
- 每位存活玩家按顺序发言一次，通常限时 60-90 秒。
- 发言内容：声称身份、分享信息、指控他人、为自己辩护等。

### 7.5 投票阶段

1. 发言结束后，警长进行归票总结。
2. 所有存活玩家同时投票（指向目标或举号码牌）。
3. 警长票数 = 1.5，其余每人 = 1。可以弃票。
4. 得票最高者被放逐出局，可发表遗言。

### 7.6 放逐投票平票处理

- 两人或多人得票相同为最高 → 进入 PK 发言。
- PK 玩家各发言一次，PK 台下玩家投票（PK 台上玩家不能投票）。
- PK 复投仅按"有投票权的台下玩家每人 1 票"计票；警长 1.5 票和归票 +0.5 在 PK 复投中均不生效。
- 若再次平票 → 当天平安日，无人出局，直接进入夜晚。

## 8. 后续夜晚流程

第二夜及以后的夜晚流程与第一夜相同，区别在于：

- 已死亡的角色不再行动（但法官仍假唤醒）。
- 女巫若解药已用完，法官不再展示被刀者。
- 守卫不能守护与上一夜相同的目标。
- 若使用狼美人板子，夜间顺序始终按 4.9 的唤醒顺序执行。

---

## 9. 后续白天流程

第二天及以后的白天流程：

1. 公布死讯（第二夜起死亡玩家无遗言）。
2. 发言（警长决定顺序，或按规则确定）。
3. 投票（与第一天相同）。
4. 放逐出局者发遗言。

注意：不再有警长竞选环节。

## 10. 死亡结算规则

### 10.1 夜间死亡结算

夜间行动采用"先收集指令，再按优先级结算"。法官在白天统一公布结果。

说明："同时结算"仅指白天统一公开结果，不代表没有内部优先级。

结算优先级与逻辑：

1. 收集夜间指令（守卫守护、狼人刀人、狼美人魅惑、女巫用药、预言家查验）。
2. 结算狼刀主死亡：
  - 守卫守护 + 女巫解药命中同一被刀目标 → 同守同救，目标死亡。
  - 仅守护或仅解药命中被刀目标 → 目标存活。
  - 守护和解药均未命中被刀目标 → 目标死亡。
3. 结算女巫毒药死亡（守卫无法抵挡毒药）。
4. 结算狼美人殉情死亡（若本轮满足触发条件）。
5. 同一玩家若出现多重死因，按最终死因判定：毒药/殉情优先于狼刀。
6. 若猎人死亡且最终死因为可开枪死因，则按号码从小到大依次结算开枪。
7. 合并当夜最终死亡名单（去重），按号码从小到大在白天公布。

### 10.2 白天死亡结算

- 投票放逐：得票最高者出局。
  - 若为白痴 → 翻牌免死，失去投票权。
  - 若为猎人 → 可以开枪。
- 猎人开枪带走的目标立即出局。
- 自爆：狼人自爆立即出局，结束当天。
- 若白天死亡者为狼美人，立即结算被魅惑者殉情出局，再进行胜利条件检查。

### 10.3 死亡公布顺序

- 夜间多人死亡：按号码从小到大公布，与实际死亡原因顺序无关。
- 这样做是为了不泄露死因信息（无法根据公布顺序推断谁是被刀、谁是被毒）。

## 11. 胜利条件检查时机

胜利条件在以下时间点检查：


| 检查点             | 说明                                |
| --------------- | --------------------------------- |
| 夜间行动结算后         | 按 10.1 的统一结算顺序检查；若触发"狼刀在先"短路则立即结束 |
| 白天放逐后           | 放逐出局后检查                           |
| 白天连锁死亡后（如狼美人殉情） | 连锁死亡全部完成后检查                       |
| 猎人开枪后           | 猎人带走玩家后检查                         |
| 狼人自爆后           | 自爆出局后检查                           |


通用原则：任何复合事件都应先完成该事件的全部连锁死亡，再执行胜利条件检查。

### 11.1 狼刀在先原则（关键）

在屠边模式下，若满足以下条件：

- 狼刀主死亡结算后已达成狼人胜利条件；
- 且该胜利由狼刀直接形成（含同守同救导致被刀目标死亡）；

则立即判定狼人胜利，后续夜间动作不再结算：

- 游戏立即结束，狼人获胜。
- 女巫的毒药、猎人的开枪等后续行动均不结算。
- 一旦触发该短路，10.1 第 5-6 步不再适用。

举例：场上剩 1 村民 + 1 狼人（屠边模式，所有神职已出局）。
狼人刀杀最后的村民 → 立即判定狼人胜利。
即使女巫同夜毒杀该狼人，也不改变结果。

## 12. 角色交互与边界情况

### 12.1 同守同救（守卫 + 女巫解药 = 目标死亡）

- 被狼刀的目标同时被守卫守护和女巫解药救助 → 两种保护抵消，目标死亡。
- 这是为了防止好人阵营双重保护过于强势。

### 12.2 守卫 vs 女巫毒药

- 守卫的守护只能抵挡狼刀。
- 守卫守护不能抵挡女巫毒药。
- 即使守卫守护了被毒者，该玩家仍然死亡。

### 12.3 猎人被毒 = 不能开枪

- 女巫毒药杀死猎人 → 猎人开枪技能失效。
- 法官在夜间会告知猎人其开枪状态。
- 猎人被狼刀杀、被投票放逐 → 可以开枪。

### 12.4 白痴翻牌后的状态

- 失去投票权，保留发言权。
- 仍可被狼人刀杀（正常死亡）。
- 仍可被女巫毒杀（正常死亡）。
- 仍可被猎人带走（正常死亡）。
- 可持有警长身份，但翻牌后在任何投票环节均不能投票（票数为 0）。
- 白痴翻牌后若持有警徽，仅保留发言顺序决定权与警徽移交/撕毁权，不享有归票加票（+0.5）。

### 12.5 自爆规则

#### 允许自爆的时机：

- 警长竞选发言环节
- 警长竞选退水环节
- 白天放逐投票发言环节
- 平票 PK 发言环节不能自爆

补充限制：

- 在所有允许自爆的发言类阶段（警长竞选发言、退水、放逐发言）中，仅允许玩家在轮到自己发言时宣布自爆，不得打断他人或跨轮次抢断。

#### 自爆效果：

- 自爆狼人立即出局。
- 当天白天阶段立即结束，不进行投票。
- 自爆无遗言（部分规则给 30 秒遗言，开局前确认）。
- 完成连锁死亡与胜利检查后进入夜晚。

#### 自爆吞警徽：

- 第一天警上阶段狼人自爆 → 本轮警长竞选取消。
- 第一天 + 第二天连续自爆 → 警徽永久流失，本局无警长。

### 12.6 骑士决斗交互

- 决斗狼人成功 → 狼人出局，白天立即结束进入夜晚，当天不投票。
- 决斗好人失败 → 骑士出局，白天继续正常发言投票流程。
- 决斗功能狼（白狼王/狼美人）→ 功能狼技能失效，直接出局。
例：狼美人被决斗出局，当晚魅惑不触发。
- 骑士决斗成功后，效果等同于自爆（结束白天），但不算自爆（不触发吞警徽）。

### 12.7 白狼王自爆带人交互

- 白狼王自爆带走猎人 → 猎人不能开枪（非正常死亡）。
- 白狼王自爆带走白痴 → 白痴不能翻牌（非投票放逐）。
- 白狼王自爆带走警长 → 先完成复合死亡并检查胜利；若游戏未结束，再由法官单独询问警徽去向。
- 白狼王自爆 = 狼人自爆的所有效果 + 额外带走一人；但若白狼王为场上最后一只狼，则不得发动带人。
- 白狼王自爆结算顺序：白狼王出局 → 带走目标出局 → 胜利检查 →（若未结束且警长死亡）警徽处理 → 进入夜晚。

### 12.8 狼美人殉情交互

- 狼美人死亡时，当前有效魅惑目标立即殉情出局；夜间死亡计入当夜结算，白天死亡在该事件后立即结算。
- 殉情的猎人 → 不能开枪。
- 殉情的白痴 → 不能翻牌。
- 殉情的警长 → 可移交/撕毁警徽。
- 狼美人被骑士决斗出局 → 魅惑不触发，被魅惑者不殉情。
- 狼美人夜间被狼队自刀 → 不允许（狼美人不能被自刀）。

### 12.9 警长死亡处理

- 警长因任何原因死亡时，可以：
  - 移交警徽给任意存活玩家（该玩家成为新警长），或
  - 撕毁警徽（本局不再有警长）。
- 移交/撕毁决定通常在警长发表遗言时做出。
- 若警长死亡场景无遗言（如夜间死亡、殉情、被白狼王带走），由法官立即单独询问。
- 若该死亡已即时触发游戏结束（如满足即时胜利判定），则不再进行警徽处理。

### 12.10 警徽流战术

- 预言家当选警长后，公开声明查验顺序（例："我今晚查 5 号，明晚查 8 号"）。
- 若预言家被刀，根据警徽流顺序，下一位被查验者的身份可被间接推断。
- 预言家死亡时将警徽移交给已验证的好人，保证好人阵营信息链不断。

## 13. 信息可见性规则

### 13.1 各角色的已知信息


| 角色  | 已知信息                      |
| --- | ------------------------- |
| 村民  | 仅知道自己的身份                  |
| 狼人  | 知道自己的身份 + 所有狼人队友的身份（含功能狼） |
| 白狼王 | 同普通狼人（知道所有狼队成员）           |
| 狼美人 | 同普通狼人 + 自己每晚的魅惑目标         |
| 预言家 | 知道自己的身份 + 每晚查验的一名玩家的阵营    |
| 女巫  | 知道自己的身份 + 每晚被刀者是谁（解药未用时）  |
| 猎人  | 知道自己的身份 + 每晚自己的开枪状态       |
| 守卫  | 知道自己的身份 + 自己每晚的守护目标       |
| 骑士  | 仅知道自己的身份                  |
| 白痴  | 仅知道自己的身份                  |


### 13.2 公开信息

- 每天白天公布的死亡名单（不公布死因）。
- 所有玩家的白天发言内容。
- 投票结果。
- 警长是谁、警徽移交情况。
- 白痴翻牌（公开白痴身份）。
- 猎人开枪（公开猎人身份和射杀目标）。

### 13.3 保密信息

- 每位玩家的身份牌（除非翻牌/出局公示）。
- 夜间具体死因（狼刀/毒药/同守同救）不公布，只公布"死亡"。
- 预言家的查验结果（由预言家自行决定是否在白天公开声称）。
- 守卫的守护目标。
- 女巫的用药情况。

### 13.4 出局玩家信息公示（可配置）

本规则由开局配置决定，分为两种变体：

- **明牌局（identity_reveal = always）：** 玩家出局时，法官翻开其身份牌向所有人展示。所有人可以看到出局者的真实身份。信息更透明，戏剧冲突更强。
- **暗牌局（identity_reveal = never）：** 玩家出局后不公开身份，只有法官知道。信息博弈更深，推理难度更高，为竞技标准。

AI 引擎应在 `game_config` 中记录该参数，并在开局前向玩家公告。

## 15. 胜利模式：屠边与屠城

### 15.1 屠边（主流竞技）

**狼人胜利条件：** 所有神职出局 或 所有村民出局。

**策略特点：**

- 村民应伪装成神职吸收狼刀（挡刀），分为明挡刀（强势站票表态）和暗挡刀（模糊发言误导）。
- 神职有时需隐藏身份，不宜过早暴露。
- 狼人需准确判断屠神还是屠民。
- 围绕身份隐藏与推理产生丰富的策略博弈。
- 抿身份（判断玩家真实身份类型）是核心技巧。

### 15.2 屠城（经典休闲）

**狼人胜利条件：** 所有好人阵营成员出局（神职 + 村民）。

**策略特点：**

- 神职通常可以直接亮身份，暴露成本较低。
- 村民战略地位较低。
- 规则更简单，策略深度不如屠边。

## 16. 速查表

### 16.1 夜晚行动速查


| 角色  | 行动               | 限制                     |
| --- | ---------------- | ---------------------- |
| 守卫  | 守护 1 名玩家         | 不能连续两晚守同一人；只挡狼刀不挡毒     |
| 狼人  | 刀杀 1 名玩家（或空刀/自刀） | 需全体统一，否则空刀；狼美人不能自刀     |
| 狼美人 | 魅惑 1 名玩家         | 狼美人死亡时被魅惑者殉情；不能空过      |
| 女巫  | 用解药 / 用毒药 / 不用药  | 各限一次；同晚不可双用；解药用完不再知被刀者 |
| 预言家 | 查验 1 名玩家阵营       | 每晚一人；不可查死人和自己          |


### 16.2 白天行动速查


| 行动    | 执行者       | 效果                                  |
| ----- | --------- | ----------------------------------- |
| 发言    | 所有存活玩家    | 声称身份、指控、分析、辩护                       |
| 投票    | 所有存活玩家    | 放逐得票最高者                             |
| 警长归票  | 警长        | 被归票者 +0.5 票；警长自身投票算 1.5 票（PK 复投不生效） |
| 自爆    | 狼人（狼美人除外） | 仅限警长竞选发言/退水、放逐发言阶段且仅本人发言轮次可发动；结算后入夜 |
| 猎人开枪  | 猎人死亡时     | 带走 1 名玩家（被毒时无效）                     |
| 白痴翻牌  | 被投票放逐时    | 免死，但永久失去投票权                         |
| 骑士决斗  | 白天发言阶段    | 决斗狼人→狼人出局进夜晚；决斗好人→骑士出局              |
| 白狼王自爆 | 白狼王       | 时机同自爆规则（PK/投票/遗言阶段禁止）；完成胜利检查与必要警徽处理后入夜 |


### 16.3 遗言规则速查


| 死亡原因         | 遗言             |
| ------------ | -------------- |
| 第一夜死亡（任意死因） | 有              |
| 第二夜及以后夜间死亡   | 无              |
| 白天投票放逐       | 有              |
| 被猎人开枪带走      | 无              |
| 狼人自爆         | 视规则（部分规则 30 秒） |


### 16.4 死亡触发技能速查


| 角色  | 触发条件    | 技能     | 失效条件                          |
| --- | ------- | ------ | ----------------------------- |
| 猎人  | 死亡时     | 开枪带走一人 | 被毒死、殉情、被白狼王带走                 |
| 白痴  | 被投票放逐时  | 翻牌免死   | 被刀杀、被毒、被猎人枪杀、殉情、被白狼王带走        |
| 骑士  | 白天主动发动  | 决斗一人   | 仅限白天发言阶段；不可在警选/平票PK/遗言时使用     |
| 白狼王 | 白天自爆时   | 带走一人   | 仅自爆触发；被放逐/被刀等方式出局不触发；最后一狼时不触发 |
| 狼美人 | 死亡时（被动） | 被魅惑者殉情 | 被骑士决斗出局时不触发                   |
